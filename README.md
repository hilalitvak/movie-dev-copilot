# 🎬 Movie Development Copilot
AI-powered production analysis tool that:
- Finds comparable films (comps)
- Estimates ROI benchmarks
- Performs similarity-based retrieval
- Supports future RAG + Pinecone integration

Built with:
- FastAPI
- TF-IDF retrieval
- Pandas data processing
- Kaggle "The Movies Dataset"
- (Upcoming) Pinecone + LLM integration
